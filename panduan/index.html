2

Menu Kasir (POS)

Menu utama untuk melayani pelanggan dengan cepat dan akurat.

Cara Bertransaksi

  1. 1Cari produk menggunakan tombol 🔍 di pojok kanan atas, atau scan barcode dengan ikon 📷
  2. 2Ketuk produk untuk menambahkan ke keranjang
  3. 3Cek keranjang dengan menekan tombol Keranjang di bagian bawah
  4. 4Ketuk Bayar untuk melanjutkan ke pembayaran

Layar Pembayaran

  • Pilih metode: Tunai, Transfer, atau Utang
  • Tunai — masukkan jumlah uang yang diterima; kembalian dihitung otomatis
  • Transfer — pastikan informasi rekening bank sudah diatur di Profil → Informasi Toko
  • QRIS — pastikan QR code sudah diunggah di Profil → Informasi Toko
  • Tekan Bayar untuk menyelesaikan transaksi
💡 Tips: Tombol cepat nominal (Rp 10.000 s/d Rp 1.000.000) tersedia untuk mempercepat input uang tunai.
3

Menu Produk

Kelola semua produk toko Anda — tambah, edit, hapus, dan filter dengan mudah.

Menambah Produk Baru

  1. 1Ketuk ikon ➕ di pojok kanan atas
  2. 2Isi informasi produk (lihat kolom di bawah)
  3. 3Ketuk Simpan Produk
Field Keterangan
Nama Produk Wajib diisi
Harga Jual Wajib diisi
Harga Modal Opsional — digunakan untuk laporan keuntungan
Stok & Stok Minimum Untuk pengecekan ketersediaan barang
Barcode Opsional — bisa di-scan langsung dari kamera
Kategori & Status Untuk pengelompokan dan filter
Foto Produk Ketuk area gambar untuk memilih dari galeri

Edit & Hapus Produk

  • Ketuk produk yang ingin diubah untuk membuka form edit
  • Scroll ke bawah, ketuk Hapus untuk menghapus produk

Filter & Urutkan

  • Gunakan ikon filter 🔽 untuk mengurutkan berdasarkan nama, harga, atau stok
  • Gunakan chip kategori di bagian atas untuk menyaring produk per kategori
4

Menu Riwayat

Lihat seluruh transaksi yang telah dilakukan, lengkap dengan ringkasan otomatis dan detail per transaksi.

Filter Transaksi

  • Ketuk tombol Filter untuk memilih rentang tanggal: Hari Ini, Kemarin, 7 Hari, Bulan Ini, atau Bulan Lalu
  • Filter juga bisa berdasarkan metode pembayaran: Tunai, Transfer, atau Utang

Ringkasan Otomatis

🧾

Total Transaksi

Jumlah transaksi dalam periode yang dipilih

💰

Total Pendapatan

Akumulasi nilai jual seluruh transaksi

📈

Pendapatan Bersih

Pendapatan dikurangi harga modal produk

🏦

Per Metode

Rincian per metode pembayaran (Tunai / Transfer / Utang)

Detail Transaksi

Ketuk salah satu transaksi untuk melihat detail item, metode pembayaran, dan kembalian. Dari halaman detail Anda dapat:

  • Edit metode atau jumlah pembayaran
  • Cetak / bagikan struk
  • Hapus transaksi (stok akan otomatis dikembalikan)
5

Profil & Pengaturan

Informasi Toko

Isi data toko yang akan tampil di struk: nama toko, alamat, nomor telepon, footer struk, serta informasi bank untuk pembayaran transfer.

Kategori Produk

Buat dan kelola kategori untuk mengelompokkan produk. Kategori bisa diaktifkan, dinonaktifkan, atau diurutkan ulang.

Pengaturan Aplikasi

Pengaturan Keterangan
Nama Perangkat Nama kasir / perangkat ini
Stok Tidak Terbatas PRO Abaikan pengecekan stok
Auto Pay Langsung bayar tanpa layar konfirmasi
Auto Print PRO Cetak struk otomatis setelah transaksi
Scanner Hardware PRO Aktifkan scanner barcode USB / Bluetooth
Format Nota Pilih format struk yang dibagikan
Format Struk Sesuaikan prefix, tanggal, dan panjang nomor struk

Pengaturan Printer

Hubungkan printer thermal via WiFi, Bluetooth, atau USB untuk mencetak struk fisik. Pilih ukuran kertas 58mm atau 80mm sesuai printer Anda.

6

Kasir Bersama PRO

Hubungkan beberapa perangkat dalam satu jaringan WiFi yang sama agar bisa bekerja bersama secara real-time.

Mengaktifkan Perangkat Utama

  1. 1Buka Profil → Kasir Bersama
  2. 2Pilih tab Utama
  3. 3Ketuk Aktifkan
  4. 4Catat Kode Sambungan dan Alamat Utama yang muncul

Menghubungkan Perangkat Lain

  1. 1Di perangkat kedua, buka Profil → Kasir Bersama
  2. 2Pilih tab Hubungkan
  3. 3Masukkan Alamat Utama dan Kode Sambungan
  4. 4Ketuk Hubungkan

Mode Perangkat Klien

🛒

Keranjang

Monitor keranjang belanja secara real-time

📷

Scanner

Scan barcode untuk menambah produk ke keranjang

🔎

Cek Stok

Cek stok produk via barcode

📊

Mutasi Stok

Tambah / kurangi stok produk

📈

Laporan

Lihat laporan penjualan real-time

💳

Kasir

Operasikan kasir mandiri dari perangkat ini

⚠️ Penting: Semua perangkat harus terhubung ke WiFi yang sama. Biarkan aplikasi tetap terbuka di perangkat utama.
7

Backup & Restore

Disarankan melakukan backup secara berkala, terutama sebelum update atau reset data.

Cara Backup

  1. 1Buka Profil → Backup & Restore
  2. 2Isi password backup
  3. 3Pilih rentang waktu data yang ingin dibackup
  4. 4Ketuk Backup Data — file akan tersimpan dan bisa dibagikan
  5. 5Simpan file di cloud (Google Drive, email, dll.)
💡 Tips: Simpan password backup di tempat yang aman. File backup tidak bisa dipulihkan tanpa password yang benar.

Cara Restore

  1. 1Buka Profil → Backup & Restore
  2. 2Masukkan password yang sama saat backup
  3. 3Ketuk Restore Data dan pilih file backup

CSV Export & Import

Ekspor data produk atau transaksi dalam format CSV untuk dibuka di Excel atau Google Sheets. Tersedia juga fitur import produk dari file CSV.

8

Lisensi Pro

Fitur PRO membuka kemampuan tambahan: Kasir Bersama, Stok Tidak Terbatas, Auto Print, Scanner Barcode Hardware, dan lainnya.

Cara Aktivasi

  1. 1Buka Profil → Aktifkan Lisensi Pro
  2. 2Masukkan token dengan format KASIR-XXXX-XXXX-XXXX
  3. 3Ketuk Aktivasi
  4. 4Aplikasi akan restart otomatis setelah berhasil
🛒 Cara Mendapatkan Token: Hubungi admin via WhatsApp atau kunjungi halaman harga di dalam aplikasi.
